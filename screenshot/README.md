# Node截图服务

### npm run dev

ts-node 直接运行，并监听文件修改自动热重启

### npm run build

使用 webpack 打包文件

### npm run serve (不使用)

webpack/tsc 编译 ts，supervisor/pm2 监听编译后文件变动重启服务，gulp 自动化任务
